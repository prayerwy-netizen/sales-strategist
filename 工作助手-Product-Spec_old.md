# 工作助手 Product Spec

## 产品说明

这是一个面向初创企业销售和品牌负责人的个人工作助手 App。用户每天通过语音或文字向助手汇报工作进展、客户跟进情况、品牌活动计划等内容，助手自动记录、整理，并基于历史数据给出下一步行动建议。帮助用户在同时管理 5-15 个项目/客户时保持清晰的工作节奏，不遗漏重要事项。

## 功能需求

- 记录工作：支持语音和文字两种方式输入每日工作内容，自动归类到对应项目/客户
- 智能整理：将零散的工作记录自动整理成结构化的项目进展、客户状态
- 任务提醒：每日早晨推送今日待办，包括需要跟进的客户、即将到期的任务
- 策略建议：根据客户跟进历史和项目进展，给出下一步行动建议
- 定期复盘：每周/每月自动生成工作总结，分析完成情况和问题
- 项目管理：支持创建、归档项目，按项目维度查看所有相关记录
- 客户管理：记录客户信息、沟通历史、签约状态
- 数据导出：导出客户清单、项目进度、工作记录为表格文件
- 报告生成：自动生成周报/月报，可直接用于汇报

## UI 布局

### 整体布局
底部 Tab 导航，共 4 个主要页面：首页、项目、客户、我的。

### 首页
- 顶部：日期显示 + 每日问候语
- 今日待办卡片区：
  - 待办事项列表，按优先级排序
  - 每项显示：事项内容、关联项目/客户、截止时间
  - 点击可标记完成或查看详情
- AI 建议卡片：
  - 显示 1-3 条今日重点建议
  - 如「张总已3天未联系，建议今天跟进」
- 底部固定：
  - 圆形「+」按钮，点击展开快速记录入口
  - 展开后显示：语音记录（麦克风图标）、文字记录（键盘图标）

### 快速记录浮层
- 语音模式：
  - 大麦克风按钮，按住说话
  - 实时显示语音转文字内容
  - 底部：取消、发送按钮
- 文字模式：
  - 多行输入框，placeholder「记录今天的工作...」
  - 智能关联：输入时自动识别提到的项目/客户名，显示标签
  - 底部：关联项目下拉、发送按钮

### 项目页
- 顶部：搜索框 + 筛选（进行中/已归档）
- 项目列表：卡片式，竖向排列
  - 每张卡片：项目名、状态标签、最近更新时间、进度条
  - 点击进入项目详情
- 右上角「+」：新建项目

### 项目详情页
- 顶部：项目名、状态、编辑按钮
- Tab 切换：动态、任务、文件
- 动态 Tab：时间线形式展示该项目所有工作记录
- 任务 Tab：该项目的待办事项列表
- 底部：快速添加记录按钮

### 客户页
- 顶部：搜索框 + 筛选（跟进中/已签约/已流失）
- 客户列表：头像 + 名称 + 公司 + 最近联系时间
- 点击进入客户详情

### 客户详情页
- 顶部：客户名、公司、状态标签
- 基本信息卡片：联系方式、来源、负责项目
- 沟通记录时间线：所有与该客户相关的记录
- 底部：快速添加沟通记录、拨打电话

### 我的页
- 头像、用户名
- 功能入口列表：
  - 周报/月报：查看和导出报告
  - 数据导出：导出客户/项目数据
  - 消息提醒设置：设置每日提醒时间
  - 归档项目：查看已归档项目
  - 帮助与反馈

## 用户使用流程

### 日常记录
1. 打开 App，首页显示今日待办和 AI 建议
2. 点击底部「+」按钮
3. 选择语音或文字方式
4. 语音：按住麦克风说「今天和张总通了电话，他对报价有疑问，需要重新做一版方案」
5. 助手自动识别关联到「张总」客户和相关项目
6. 确认后发送，记录自动归档

### 查看建议
1. 每天早上收到推送「今日工作提醒」
2. 打开查看 3 件优先事项
3. 点击某条建议，查看详细分析和建议行动
4. 完成后标记已处理

### 生成报告
1. 进入「我的」-「周报/月报」
2. 选择时间范围
3. 助手自动生成报告，包含：本周完成事项、客户跟进情况、项目进展、下周计划
4. 预览后可导出为 PDF 或分享

### 复盘分析
1. 周末收到「本周复盘」推送
2. 查看本周工作数据：跟进客户数、完成任务数、新增项目等
3. 查看 AI 分析的问题和改进建议
4. 参考建议调整下周计划

## 补充说明

| 项目 | 说明 |
|------|------|
| 项目状态 | 进行中 / 暂停 / 已完成 / 已归档 |
| 客户状态 | 初次接触 / 跟进中 / 方案阶段 / 谈判阶段 / 已签约 / 已流失 |
| 任务优先级 | 高（今日必做）/ 中（本周完成）/ 低（有空再说）|
| 提醒时间 | 默认每日 8:00 推送，可自定义 |
| 数据导出格式 | Excel (.xlsx) / CSV |
| 报告格式 | PDF / Markdown |
| 数据存储 | 按项目周期保存，项目归档后数据保留可查询 |

## 应用场景

- **早晨开工**：小王打开 App，查看今日待办，发现有3个客户需要跟进，按提醒顺序逐一联系
- **外出拜访后**：刚和客户聊完，在出租车上用语音快速记录沟通要点和客户反馈
- **周会准备**：周五下午，用「生成周报」功能一键生成本周工作总结，直接发给老板
- **客户分析**：打开某个重要客户的详情页，查看所有沟通历史，思考破局策略
- **月度复盘**：月底查看月报，分析哪些客户转化了、哪些流失了、原因是什么
