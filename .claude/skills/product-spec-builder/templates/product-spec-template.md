# Product Spec 输出模板

本模板用于生成结构完整的 Product Spec 文档。生成时按照此结构填充内容。

---

## 模板结构

**文件命名**：<产品名称>-Product-Spec.md

---

## 产品说明
<一段话：这是什么、解决什么问题、给谁用、核心价值>

## 功能需求
<功能列表，动词开头，简明扼要>

## UI 布局
<描述整体布局结构和各区域的详细设计，需要包含：>
- 整体是什么布局（几栏、比例、固定元素等）
- 每个区域放什么内容
- 控件的具体规范（位置、尺寸、样式等）

## 用户使用流程
<分步骤描述用户如何使用产品，可以有多条路径（如快速上手、进阶使用）>

## 补充说明
<如有需要，用表格说明选项、状态、逻辑等>

## 应用场景
<列举 3-5 个具体场景：谁、在什么情况下、怎么用>

## 完整示例

以下是一个「视频提示词生成器」的 Product Spec 示例，供参考：

```markdown
## 产品说明

这是一个帮助用户快速生成专业级 AI 视频提示词的工具。用户只需要用简单的语言描述想要的画面（甚至只说一个大概想法），AI 就会根据最佳实践，自动补全主体描述、动作、场景、镜头运动、灯光、风格、音效等专业要素，输出可以直接复制使用的完整提示词。支持文生视频和图生视频两种模式，支持一次生成多个风格变体。

## 功能需求

- 智能补全：用户输入模糊描述时，AI 自动推荐并补全缺失的要素（镜头、灯光、音效等）
- 图片理解：图生视频模式下，AI 分析上传的图片，识别主体、场景、色调、风格，并据此生成匹配的提示词
- 批量生成：一次输出 3-5 个不同风格/角度的提示词变体供选择
- 双模式支持：区分「文生视频」和「图生视频」两种场景，生成对应格式的提示词
- 一键复制：每个生成的提示词都有复制按钮
- 中英双语：用户可用中文输入，输出英文提示词

## UI 布局

### 整体布局
左右两栏布局，左侧输入区占 35%，右侧输出区占 65%。

### 左侧 - 输入区
- 顶部：模式切换 Tab（文生视频 / 图生视频）
- 文生视频模式：
    - 主输入框：多行文本框，铺满区域宽度，placeholder「描述你想要的画面...」
- 图生视频模式：
    - 图片上传区：支持拖拽或点击上传，显示图片预览
    - 文本输入框（较小）：补充说明希望图片如何「动起来」
- 可展开的「高级选项」面板，默认收起，包含：
    - 镜头类型：下拉选择（特写 / 中景 / 远景 / 跟踪镜头），默认「自动」
    - 镜头运动：下拉选择（推 / 拉 / 摇 / 移 / 固定），默认「自动」
    - 灯光风格：下拉选择（自然光 / 黄金时刻 / 霓虹 / 逆光），默认「自动」
    - 视觉风格：下拉选择（电影感 / 动画 / 复古 / 赛博朋克），默认「电影感」
- 底部：
    - 生成数量：下拉选择（1条 / 3条 / 5条），默认「3条」
    - 「生成提示词」主按钮，靠右对齐，醒目样式

### 右侧 - 输出区
- 卡片式展示生成结果，竖向排列，可滚动
- 每张卡片包含：
    - 风格标签（如「电影感」「动感」）
    - 完整提示词文本
    - 「复制」按钮，点击后显示「已复制」

## 用户使用流程

### 快速生成（文生视频）
1. 默认进入「文生视频」模式
2. 在主输入框用中文简单描述想要的画面，比如「一个女孩在雨中奔跑」
3. 点击「生成提示词」
4. 在右侧查看 AI 生成的 3 个提示词变体
5. 选择喜欢的，点击「复制」
6. 粘贴到 AI 视频工具使用

### 图生视频
1. 切换到「图生视频」Tab
2. 上传一张图片（风景照、人物照、产品图等）
3. 在文本框补充说明希望的动态效果，如「云慢慢飘动，镜头缓缓上升」
4. （可选）展开高级选项微调
5. 点击「生成提示词」
6. 复制后粘贴到 AI 视频工具的图生视频功能使用

## 补充说明

| 选项 | 可选值 | 说明 |
|------|--------|------|
| 镜头类型 | 自动 / 特写 / 中景 / 远景 / 跟踪镜头 | 「自动」时 AI 根据内容推荐 |
| 镜头运动 | 自动 / 推 / 拉 / 摇 / 移 / 固定 | 「自动」时 AI 根据内容推荐 |
| 生成数量 | 1条 / 3条 / 5条 | 一次生成多个风格变体供选择 |

## 应用场景

- **创意初期**：用户只有模糊想法（如「科技感的产品展示」），需要 AI 帮忙具象化成完整画面
- **批量创作**：短视频创作者需要快速产出多条不同风格的视频脚本
- **学习提升**：新手通过对比自己的描述和 AI 补全后的结果，学习专业的提示词写法
- **商业广告**：为产品/品牌快速生成多个视觉方案的提示词进行测试
- **静态图激活**：设计师/摄影师将已有的静态作品转化为动态视频

## Google AI Studio 能力配置

**在 Builder 中请勾选以下能力：**

| 能力名称 | 用途说明 |
|---------|---------|
| Gemini Intelligence in Your App | 用于智能补全提示词、分析用户输入、生成多个风格变体 |
| Analyze Images | 用于图生视频模式下分析上传的图片，识别主体、场景、色调 |
| Fast AI Responses | 用于快速响应用户输入，提供流畅的交互体验 |
```

## 写作要点

1. **产品说明**：一段话说清楚是什么、解决什么问题、给谁用
2. **功能需求**：动词开头，简明扼要，不超过 10 条
3. **UI 布局**：
   - 先写整体布局（几栏、比例）
   - 再逐个区域描述内容
   - 控件要具体：下拉框写出所有选项和默认值，按钮写明位置和样式
4. **用户流程**：分步骤，可以有多条路径
5. **补充说明**：用表格，适合解释选项、状态、逻辑
6. **应用场景**：具体的人 + 具体的情况 + 具体的用法
7. **AI 能力配置**：从 reference.md 中选择，说明在本产品中的具体用途
